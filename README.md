# Spring Grocery List POC

This is a proof-of-concept for a grocery list application built with Spring Boot.

## Tech Stack

*   **Java 25**: The application is built using Java 25.
*   **Spring Boot 4.0.1**: The application is built using the Spring Boot framework.
*   **H2 Database**: An in-memory database is used for this POC.
*   **Maven**: Dependency management is handled by Maven.
*   **OpenAPI (Swagger)**: API documentation is provided by OpenAPI (Swagger).
*   **JUnit 5**: Unit and integration tests are written using JUnit 5.

## How to Run

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/spring-grocery-list-poc.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd spring-grocery-list-poc
    ```
3.  Run the application using Maven:
    ```bash
    ./mvnw spring-boot:run
    ```
4.  The application will be available at `http://localhost:8080`.

## API Documentation

The API documentation is available at `http://localhost:8080/swagger-ui.html`.
